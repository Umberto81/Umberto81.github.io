MERN App

Cash Drawer - till Web application.

A MERN application, built with react hooks, express, and hosted on mongoDB. This app provides all the necessary functions for registering and calculating transactions at a point of sale. Products with related images and details are stored on a MongoDb database. Among the advanced features, there is a login authentication, a private routing service (with all the needed API calls) and the ability to update and modify the database. (work in progress ..).
